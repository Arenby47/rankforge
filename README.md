RankForge is a full-stack web application designed to track and analyze competitive gaming performance.
The platform allows players to record their match results, monitor performance statistics, and view their gameplay history through a secure authentication system.

The system uses JWT-based authentication to ensure that each user's data is protected and accessible only to authorized players. Users can log in, record match details, and view their performance metrics such as win rate and K/D ratio.

Features

User Registration and Login

Secure Authentication using JWT

Password Encryption using bcrypt

Add Match Results

View Match History

Player Performance Statistics

Profile Page

Frontend Dashboard Interface

RESTful API Architecture
